## REST endpoints
### Quests
- GET /quests — отримати всі квести
- GET /quests/{id} — отримати квест за id
- POST /quests — створити квест
- PUT /quests/{id} — оновити квест
- DELETE /quests/{id} — видалити квест

### Teams
- GET /teams — отримати всі команди
- GET /teams/{id} — отримати команду за id
- POST /teams — створити команду
- PUT /teams/{id} — оновити команду
- DELETE /teams/{id} — видалити команду

### Registrations
- GET /registrations — отримати всі реєстрації
- GET /registrations/{id} — отримати реєстрацію за id
- POST /registrations — створити реєстрацію
- PUT /registrations/{id} — оновити реєстрацію
- DELETE /registrations/{id} — видалити реєстрацію
